# Music Video Template

A template for quickly creating music videos.Support cover setting, audio visualization, and lyrics display.

## How do I render my video?

Run this:

```console
npm run build
```

Or check out the [Remotion docs](/docs/render/). There are lots of ways to render.

## Where to get a transcript (SRT file)?

use `tools/MusicLyricApp` to get music lyric.

## Commands

**Install Dependencies**

```console
npm install
```

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Upgrade Remotion**

```console
npm run upgrade
```
